# AlphaPlugins

These are the plugins i've made for Minecraft Alpha HMod (specifically for Alpha 1.1.x), since they are very rare now. About the plugins:

## AlphaLogin
A basic login system that works on a .properties file with PBKDF2 encryption.
## AlphaEssentials
A multi-purpose plugin that is for the most of user commands, TPA, spawn protection etc.
## AlphaProtection
A protection plugin to protect player's important stuff (doors for now only).
